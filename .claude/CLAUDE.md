# `.claude/` — reserved areas

This note is for any future contributor (human or coding agent) working in this repo.

## `.claude/developer/` is reserved per-contributor scratch space

- It is **reserved for an individual contributor's coding-agent resources** — personal plans, scratch notes, throwaway prompts, local agent configs, experiment logs, anything you want.
- It is **already gitignored**, so nothing inside it is ever committed or shared. You are **free to add anything** there without affecting other contributors or the repo history.
- Put your personal/agent working files **here**, not loose elsewhere in the repo. Keep the committed tree clean.

## Everything else in `.claude/`

Other Markdown files committed at the top of `.claude/` are **shared project docs** (project context, plan, theory). Treat those as common, reviewed material — not personal scratch.
